# 2026-03-10 - 记忆系统 P0 修复

## 问题发现
用户问起 P0 bug，发现记忆系统虽然代码在主链路上，但实际没工作。

## 根本原因
1. **依赖缺失** - `faiss-cpu` 和 `sentence-transformers` 没安装
2. **异常静默吞掉** - 满屏 `except Exception: pass`，没有任何日志
3. **测试只测 happy path** - mock 绕过真实依赖，PR 测试全绿但上线就挂

## 修复内容
1. 安装依赖
2. 添加 logging 到三个文件：
   - `agentlib/runtime_engine.py`
   - `agentlib/companion_rag.py`
   - `agentlib/memory_store.py`
3. TF-IDF fallback - 避免 HuggingFace 网络依赖
4. 异常现在会记录日志，不再静默失败
