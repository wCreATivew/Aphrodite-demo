## 2026-03-09 - 项目启动

### 游戏式 AI 角色世界项目

用户正在开发一个"游戏式 AI 角色世界"项目，核心目标是构建可持续运行的交互世界。

**核心设计原则：**
1. 角色具备长期记忆、稳定人格、个体化理解
2. 关系状态会随时间演化
3. 世界内有持续状态与事件追踪
4. 类游戏的表现与互动结构

**关键边界：**
- 不是完美故事生成器
- 不是现实替代品
- 保留遗憾、因果和限制

**核心价值：** "让用户获得原本没有的沟通机会"

### ⚠️ 重要教训：避免过度开发 Agent

用户之前做过类似项目，失败原因是**过度开发代理 agent**。

**正确方向：**
- 核心不是 agent 有多强，而是世界有多真
- 把 agent 当"演员"，不是"主角"
- 追求人格一致，不是完美回答
- 追求世界可信，不是对话完美

**这次开发原则：**
- 最小可行世界 (MVW) 先行
- 每个功能问：服务于核心价值吗？
- 避免功能蔓延和技术驱动
