# 2026-03-11 - 工作时间约束 + 总包模式

## 用户时间约束
**可深度交互时段：**
- 12:00-14:00 (午休)
- 18:00-19:00 (傍晚)

**其他时间：** 用户上学，Agent 自主工作

## 工作模式
**总包模式：**
- 我 (主 Agent) 是总包，负责任务分解和协调
- 具体执行 spawn subagent 到 worktree 里做
- 使用 `scripts/setup-worktree.sh` 创建隔离工作区

## Worktree 规范
```bash
# 创建新项目
./scripts/setup-worktree.sh <项目名> "<任务描述>"

# 工作区结构
/home/creative/.openclaw/workspace/<项目名>        ← git repo
/home/creative/.openclaw/workspace/<项目名>-test   ← 工作区 (subagent 在此工作)
```

## 自主工作原则
用户不在时：
1. 在 worktree 里测试、验证
2. 不推送远程 (等用户确认)
3. 记录进展到 mission.md
4. 有问题记下来，等用户在线时问
