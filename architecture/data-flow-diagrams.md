## 1. 完整对话数据流

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              用户对话完整数据流                              │
└─────────────────────────────────────────────────────────────────────────────┘

    ┌──────────┐                              ┌──────────────────┐
    │   用户    │                              │   后端服务集群    │
    │  (前端)   │                              │    (Python)      │
    └────┬─────┘                              └────────┬─────────┘
         │                                            │
         │  ① 输入消息                                │
         │  "今天天气真好！"                           │
         │  ─────────────────────────────────────────→│
         │                                            │
         │                                            │ ② 保存用户消息
         │                                            │    ↓
         │                                            │ ┌─────────────┐
         │                                            │ │   Memory    │
         │                                            │ │   Module    │
         │                                            │ └─────────────┘
         │                                            │
         │                                            │ ③ 查询历史记忆
         │                                            │    ↓
         │                                            │ ┌─────────────┐
         │                                            │ │  LLM Service│
         │                                            │ │ (回复生成)   │
         │                                            │ └─────────────┘
         │                                            │
         │                                            │ ④ 情感分析
         │                                            │    ↓
         │                                            │ ┌─────────────┐
         │                                            │ │   Emotion   │
         │                                            │ │   Analyzer  │
         │                                            │ └─────────────┘
         │                                            │
         │  ⑤ 推送回复                                │
         │  ←─────────────────────────────────────────│
         │  {                                         │
         │    "type": "character_reply",              │
         │    "content": "是啊，阳光明媚呢！",          │
         │    "emotion": "happy",                     │
         │    "expression_id": "exp_happy"            │
         │  }                                         │
         │                                            │
         │  ⑥ 切换表情                                │
         │  ┌─────────────────┐                       │
         │  │  Live2D 渲染器   │ ← setExpression()     │
         │  │   (WebGL)       │    "exp_happy"        │
         │  └─────────────────┘                       │
         │                                            │
         │  ⑦ 推送语音流 (分块)                        │
         │  ←─────────────────────────────────────────│ ⑧ GPT-SoVITS
         │  [Audio Chunk 0]                           │    流式合成
         │  [Audio Chunk 1]                           │    ←────────┐
         │  [Audio Chunk 2]                           │             │
         │  ...                                       │             │
         │  [Audio Chunk N]                           │             │
         │                                            │             │
         │  ⑨ 播放音频                                │             │
         │  ┌─────────────────┐                       │             │
         │  │  AudioContext   │ ← play()              │             │
         │  │  (Web Audio)    │                       │             │
         │  └─────────────────┘                       │             │
         │                                            │             │
         │  ⑩ 消息上屏                               │             │
         │  ┌─────────────────┐                       │             │
         │  │  消息气泡列表    │ ← render()            │             │
         │  │  (Vue/React)    │                       │             │
         │  └─────────────────┘                       │             │
         │                                            │             │
         └────────────────────────────────────────────┘             │
                                                                    │
                               ┌────────────────────────────────────┘
                               │
                          ┌────┴─────┐
                          │ GPT-SoVITS│
                          │  TTS 服务  │
                          └──────────┘
```

## 2. WebSocket 消息时序图

```
时间轴 ──────────────────────────────────────────────────────────────────────→

前端 (Browser)              后端 (Python)                  Live2D 渲染
     │                          │                              │
     │── Connect ─────────────→│                              │
     │                          │                              │
     │←─ Auth OK ──────────────│                              │
     │                          │                              │
     │                          │                              │
     │── user_message ─────────→│                              │
     │  "你好呀"                 │                              │
     │                          │                              │
     │                          │── [处理中] ──────────────────→│
     │                          │   setExpression('thinking')   │
     │                          │                              │
     │←─ expression_update ─────│                              │
     │  {expression: 'thinking'}│                              │
     │                          │                              │
     │                          │── [生成回复]                  │
     │                          │                              │
     │←─ character_reply ───────│                              │
     │  {content: "你好！",       │                              │
     │   emotion: "happy"}      │                              │
     │                          │                              │
     │  setExpression('happy') ─→│                              │
     │                          │                              │
     │←─ motion_trigger ────────│                              │
     │  {motion: "speak"}       │                              │
     │                          │                              │
     │  triggerMotion('speak') ─→│                              │
     │                          │                              │
     │←─ audio_chunk (1) ───────│                              │
     │←─ audio_chunk (2) ───────│                              │
     │←─ audio_chunk (3) ───────│                              │
     │                          │                              │
     │  playAudio() ────────────→│                              │
     │  (流式播放)               │                              │
     │                          │                              │
     │←─ expression_update ─────│                              │
     │  {expression: 'neutral'} │                              │
     │                          │                              │
     │  setExpression('neutral')→│                              │
     │                          │                              │
     │  stopMotion('speak') ────→│                              │
     │                          │                              │
```

## 3. 表情状态机

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           表情状态转换图                                     │
└─────────────────────────────────────────────────────────────────────────────┘

                              ┌─────────────┐
                              │    IDLE     │
                              │  (neutral)  │
                              └──────┬──────┘
                                     │
                     ┌───────────────┼───────────────┐
                     │               │               │
                     ▼               ▼               ▼
            ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
            │  LISTENING  │ │  THINKING   │ │  SPEAKING   │
            │  (neutral)  │ │  (thinking) │ │  (emotion)  │
            └──────┬──────┘ └──────┬──────┘ └──────┬──────┘
                   │               │               │
                   │               │               │
                   └───────────────┴───────────────┘
                                     │
                                     ▼
                              ┌─────────────┐
                              │ TRANSITION  │
                              │  (blend)    │
                              └──────┬──────┘
                                     │
                                     ▼
                              ┌─────────────┐
                              │    IDLE     │
                              │  (neutral)  │
                              └─────────────┘


状态说明:
┌─────────────┬─────────────────────────────────────────────────────────────┐
│    状态      │                        触发条件/行为                        │
├─────────────┼─────────────────────────────────────────────────────────────┤
│    IDLE     │ 默认状态，表情为 neutral，播放 idle 动作                       │
│  LISTENING  │ 用户开始输入时进入，表情保持 neutral，等待回复                 │
│  THINKING   │ 后端处理中，显示 thinking 表情，可配合思考动作                   │
│  SPEAKING   │ 回复生成后，显示对应情感表情，播放 speak 动作，同时播放语音     │
│ TRANSITION  │ 表情切换过渡状态，使用 blend 动画                              │
└─────────────┴─────────────────────────────────────────────────────────────┘
```

## 6. 错误处理流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                            错误处理流程                                      │
└─────────────────────────────────────────────────────────────────────────────┘

    前端                          后端                          用户
     │                            │                            │
     │  ① 发送消息                  │                            │
     │  ─────────────────────────→│                            │
     │                            │                            │
     │                            │  ② 处理失败                  │
     │                            │  (LLM 超时/服务错误)          │
     │                            │                            │
     │  ③ 返回错误消息              │                            │
     │  ←─────────────────────────│                            │
     │  {                         │                            │
     │    "type": "error",        │                            │
     │    "code": "LLM_TIMEOUT",  │                            │
     │    "message": "..."        │                            │
     │  }                         │                            │
     │                            │                            │
     │  ④ 显示错误提示              │                            │
     │  ┌──────────────┐          │                            │
     │  │ ❌ 发送失败   │          │                            │
     │  │ [重试] [编辑] │          │                            │
     │  └──────────────┘          │                            │
     │                            │                            │
     │  ⑤ 用户点击重试              │                            │
     │  ─────────────────────────→│  ⑥ 重新处理                  │
     │  (重新发送原消息)            │                            │
     │                            │                            │
     │  ⑦ 成功回复                  │                            │
     │  ←─────────────────────────│                            │
     │                            │                            │

错误类型:
┌──────────────────┬─────────────────────────────────────────────────────────┐
│     错误代码      │                        处理建议                          │
├──────────────────┼─────────────────────────────────────────────────────────┤
│  LLM_TIMEOUT     │ 稍后重试，显示"角色正在思考中..."                          │
│  TTS_ERROR       │ 继续显示文本，语音可稍后补发或跳过                         │
│  WS_DISCONNECT   │ 自动重连，重连后同步未发送消息                            │
│  AUTH_FAILED     │ 跳转到登录页面                                           │
│  RATE_LIMITED    │ 显示"消息发送太频繁，请稍后再试"                           │
└──────────────────┴─────────────────────────────────────────────────────────┘
```

## 7. 网络状态处理

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          网络状态处理流程                                    │
└─────────────────────────────────────────────────────────────────────────────┘

    正常状态                        断网                        恢复连接
     │                            │                            │
     │  ┌──────────┐              │                            │
     │  │ Connected│              │                            │
     │  └──────────┘              │                            │
     │       │                    │                            │
     │       │ 网络断开             │                            │
     │       ▼                    │                            │
     │  ┌──────────┐              │                            │
     │  │Reconnecting│─────────────→│  ① 显示"连接中..."           │
     │  └──────────┘              │     ② 消息进入发送队列       │
     │       │                    │     ③ 指数退避重连            │
     │       │                    │       (1s, 2s, 4s, 8s...)    │
     │       │                    │                            │
     │       │ 重连成功            │                            │
     │       ▼                    │                            │
     │  ┌──────────┐              │                            │
     │  │ Connected│ ←─────────────│  ④ 发送队列中的消息          │
     │  └──────────┘              │  ⑤ 同步未接收的消息          │
     │       │                    │  ⑥ 恢复表情/语音状态         │
     │       │                    │                            │

重连策略:
- 最大重试次数：5 次
- 初始延迟：1 秒
- 延迟倍增：2 倍
- 最大延迟：30 秒
- 心跳检测：30 秒间隔
```
